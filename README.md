# Scroll Story — Angular + GSAP

Projeto interativo de scroll feito com Angular e GSAP.  
Cada seção da página possui animações suaves e títulos que aparecem com efeito de rolagem.

## Tecnologias usadas
- Angular 17
- TypeScript
- GSAP (ScrollTrigger)
- CSS moderno e gradientes dinâmicos
